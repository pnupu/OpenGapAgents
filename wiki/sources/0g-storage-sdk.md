## Key Points

0G Storage provides SDKs and starter kits for storing and retrieving files. The docs mention TypeScript and Go starter kits, file upload, Merkle-root calculation, download by root hash, and optional proof verification during download.

## Relevance To OpenGap

Use 0G Storage for:

- interaction traces
- source snapshots
- generated docs/examples
- validation results
- gap clusters and score history
- project memory exported from the local wiki

## Demo Artifact

The demo should upload at least one generated artifact to 0G Storage and display:

- artifact type
- root/hash
- validation status
- linked source question or trace
