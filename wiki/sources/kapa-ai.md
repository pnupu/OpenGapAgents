## What It Does

Kapa indexes technical product knowledge from docs, source code, PDFs, support tickets, community conversations, and other sources. It provides docs-site Q&A, community answering, support deflection, internal technical assistant use cases, MCP context for coding agents, and analytics.

## Relevant Claim

Kapa states that every user question is a signal, persists conversations, and provides analytics such as coverage gaps showing where documentation is missing or insufficient.

## Relevance To OpenGap

Kapa validates the need for developer-support assistants and documentation gap analytics. It is one of the closest existing commercial analogs.

## Differentiation Needed

OpenGap should not compete as another hosted docs chat widget. The project should focus on:

- open framework/tooling
- 0G-native persistent memory
- generated and validated examples
- onchain provenance of generated improvements
- plugin architecture for ecosystem agents
