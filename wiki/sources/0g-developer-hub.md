## Key Points

0G positions itself as modular infrastructure for AI applications. The Developer Hub groups the platform into four services:

- 0G Chain: EVM-compatible blockchain optimized for AI apps.
- 0G Compute: decentralized GPU marketplace for AI workloads.
- 0G Storage: high-performance storage for massive datasets.
- 0G DA: scalable data availability for any chain.

## Relevance To OpenGap

The docs support using only the components needed. For OpenGap, the core components are:

- Compute for inference, reflection, scoring, and generation.
- Storage for traces, generated artifacts, memory, and source snapshots.
- Chain for registry/proof commitments.
- DA only if we need blob availability or rollup/appchain-oriented proof flows.
