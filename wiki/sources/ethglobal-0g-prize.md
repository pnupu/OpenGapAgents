# Source: ETHGlobal Open Agents 0G Prize Page

Status: active  
Updated: 2026-04-25  
Source URL: https://ethglobal.com/events/openagents/prizes/0g

## Relevant Prize Track

Target: Best Agent Framework, Tooling & Core Extensions.

The track is for framework-level work: core extensions, improvements, forks, or new open agent frameworks deployed on 0G. The page specifically encourages infrastructure primitives for how agents are created in 2026.

## Track Ideas That Map To OpenGap

- Framework-level modules for reflection loops.
- Self-evolving agent framework that generates, tests, and integrates new skills/tools.
- Persistent memory through 0G Storage.
- Swappable memory, LLM backend, and decision-engine layers.

## Submission Requirements

Need:

- project name and short description
- contract deployment addresses
- public GitHub repo with README and setup instructions
- demo video under 3 minutes
- live demo link
- explanation of used 0G protocol features or SDKs
- team member names and contact info
- at least one working example agent
- architecture diagram strongly recommended

## Implication

OpenGap must include a working example agent built with the framework. The example agent should be the 0G Builder Agent.
