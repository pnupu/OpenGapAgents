## What It Does

GitBook offers AI-assisted documentation workflows and docs analytics. Its analytics material emphasizes that search queries and AI questions can reveal documentation gaps that may not be obvious from content analysis alone.

## Relevance To OpenGap

GitBook shows that AI-docs analytics is becoming mainstream in documentation platforms.

## OpenGap Differentiation

OpenGap should not present analytics as the core novelty. The stronger angle is an open framework loop where an agent:

- detects gaps
- generates the missing artifact
- validates it
- stores it on 0G
- registers it onchain
