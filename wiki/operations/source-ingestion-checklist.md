# Source Ingestion Checklist

Status: active  
Updated: 2026-04-25

Use this when adding new sponsor docs, SDK docs, examples, or project research.

## Checklist

- Capture source URL.
- Record retrieval date.
- Summarize only the relevant points.
- Extract concrete integration requirements.
- Add open questions.
- Link affected project/concept pages.
- Update `wiki/index.md`.
- Append to `wiki/log.md`.
