Status: active  
Updated: 2026-04-25

## Required

- Project name: OpenGap Agents or final selected name.
- Short description.
- Contract deployment address on 0G Chain.
- Public GitHub repo.
- README with setup instructions.
- Demo video under 3 minutes.
- Live demo link.
- Explanation of 0G protocol features/SDKs used.
- Team member names and contact info.
- At least one working example agent.
- Architecture diagram recommended.

## Demo Video Script

1. State the problem: docs agents answer questions but do not repair the knowledge base.
2. Show developer asking the 0G Builder Agent a question.
3. Show answer with source citations.
4. Show gap detection.
5. Show generated doc/example.
6. Show validation result.
7. Show 0G Storage root.
8. Show 0G Chain registry event/transaction.
9. End with framework extension points.
